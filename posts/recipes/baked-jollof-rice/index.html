Baked Jollof Rice Return recipe Posted September 22, 2024 Updated September 22, 2024 2 min read

I took my old [[baked-beans-and-rice|Baked Beans & Rice]] recipe and modified it using this Jollof Rice recipe I found from Ethan Cheblowski.

The result was fantastic. I served this with BBQ chicken for a meal, using some of the spice mix on the chicken. I still had plenty of leftover spice mix, which I will definitely be using again.

The standalone recipe can be found [[jollof-seasoning|here]].
