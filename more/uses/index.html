A list of things I use, and why I use them.

Software

Operating System

I use Nobara Linux (GNOME Edition) on both my desktop and laptop. It’s an opinionated fork of Fedora, with a focus on gaming and content creation. It has really good defaults and out-of-the-box experience. It works very well for my use case.

IDE for C++

My code editor for c++ projects is Clion. It has the best code completion and linting for c++ that I have ever used, I consider it better than Visual Studio which I regarded as best-in-class for a long time.

I liked it so much I purchased a license but recently a full-featured free version has been released, for non-commercial use. I’m only a little bitter about that.

IDE for Everything else

My go-to code editor for most projects is VSCodium. It’s especially well suited to web devlopment. It’s what I use to develop this site.

It is a fork of Microsoft’s VScode. It’s purpose is to be a drop-in replacement for VScode without Microsoft’s proprietary licensing or telemetry. I care about both of these things which is why I choose to use this over the original.

I try to keep the extensions that I use to a minimum, as they have been demonstrated to be a security risk. I’m pretty amateur as a web developer, so I’ve found CSS Peek to be extremely useful.

Terminal Emulator

Ghostty is a very good terminal emulator, and its currently my favorite. It’s fast, actively maintained, and has a very good default configuration. I like Alacritty as well and did use it for a while, but I switched to Ghostty simply because it integrated better with my GNOME based desktop.

Note Taking

An app called Obsidian. I first discovered it while watching this video by Wendell of Level1Techs. Notes are stored in markdown format text files, and there is a large collection of plugins and themes available for download directly in the user interface.

My note-taking is not as elaborate as Wendell’s, but Obsidian is really good even just as a markdown editor. I have found it useful to be able to create links between my notes, so I do use this feature often. I don’t have much use for the more advanced features like the canvas though. The graph view looks amazing but I don’t find it useful.

It is free to download but I loved it so much I purchased a license. I was really excited when the bases feature entered beta and I wanted to try it before its full release (beta access is limited to the paid version).

Web Browser

Firefox is the best web browser to use if you care about security, privacy, and ad-blocking. I’ve been using it since I was in high school. It was the first browser I ever saw to use tabs, a mind-boggling feature at the time (yes, I’m old).

In recent years Firefox has failed to keep up with its competition. Compared to Chrome it is slower, uses more memory, and crashes more often. Its market share has fallen to just 2.4% in 2025, with a downward trend from the year before.

Despite its flaws I still prefer it for the addon ecosystem. I consider good ad-blocking a requirement for any browser I use, as well as good privacy protections, and with addons Firefox is the only browser that suits my needs.

This is my current list of essential addons:

  • uBlock Origin - The best ad blocker I’ve ever used. Actively maintained.
  • Privacy Badger - Provides additional blocks against online tracking.
  • Decentraleyes - A ‘Local CDN Emulator’, good for both privacy and performance.

Command line Programs

The command utilities that I use. Whether its a vm, container, or bare-metal box I try to have these installed at all times.

  • micro - Text editor. Better than nano.
  • fzf - Easier to use than grep.
  • tldr - Short-form man pages.
  • tmux - For running background terminal sessions.
  • fish - Bash alternative. Good defaults.
  • btop - My preferred system monitor, otherwise very well made.

Games

Monster Hunter Series

Monster Hunter is a fantastic series of games. I’ve been a fan almost as long as the series has existed. I remember going to Blockbuster as a kid and buying Monster Hunter Freedom for my PSP. I’d walk past the box a few times before finally buying it. I was excited to become the hunter on the box, fighting a nasty looking dragon with a sword made from a skull. My favorite game in the series is World, but I also enjoy the newer titles Rise and Wilds.

The premise is straight-forward. You go on hunts in which you either kill or capture a monster. You can carve items from the monster if it was killed, and you always receive additional hunt rewards. Items obtained on the hunts can be used to craft weapons and armor. Hunting increasingly powerful monsters requires better gear, which requires powerful items that must be obtained from hunting certain monsters. This creates a satisfying loop of progression towards the end of the game.

I like to play solo as well as co-op, I highly recommend it.

Tetris

I play a lot of Tetris. I use it primarily as a way to give my mind a break if I’m feeling stressed or fatigued. I’ll also pick it up for a round if I have a bit of time to kill.

There are several Tetris variants that I particularily enjoy:

  • TGM2 - My absolute favorite. I play it nearly every day. I like that it has lock delay and floorkick gameplay mechanics.
  • Gameboy Tetris - My favorite classic Tetris. You can’t argue with 35 million unit sales.
  • Heboris Unofficial Expansion - A fan-made variant. It feels similar to TGM2, and I played it for a long time before discovering TGM2. It’s also open source, and the PS Vita port in particular is very well done.

Everyday Carry

Watch

I wear a Garmin Vivoactive 4. I bought mine used a few years ago and it has worked well for me. I like that it looks very normal, like a traditional watch. The ‘squircle’ shape popularized by Apple doesn’t look right to me. I also really don’t like the ‘bubble’ look of Google’s watches.

It has a decent software ecosystem, with apps and custom watch faces. I played around with the development tools, and even created my own custom watch face. Though I do think Garmin’s Monkey C programming language must have been created by an intern, lol.

The biggest reason I had for getting a Garmin was the transflective LCD display. It reflects light in a way similar to e-ink, which means its fully readable in well lit environments, including direct sunlight. Eventually I think OLED and then MicroLED watches will be bright enough that visibility stops being a concern, but until then I will keep recommending and using Garmin watches.

Phone

My current phone is a Google Pixel 7 Pro. I got it in 2025 and its the first ‘phablet’ style phone I’ve had in years. I enjoy the extra screen real-estate and battery life, but I do sometimes miss the useability of a one-handed phone.

Coming from a Samsung Galaxy S20 I was very interested in getting a phone with custom ROM support, which this phone most definitely does (all pixel series phones have it). The S20 is a marvel and an astounding piece of hardware, but it is completely locked down by Samsung, there is no option that I found to even gain root access. That always bothered me, seeing as I’m an absolute control freak with my computers. I might not care to tweak every last detail of my device, but it irritates me greatly if I don’t at least have the choice.

The camera is excellent, it’d be weird if it wasn’t. It also has a 120hz display like my previous Samsung, a feature that I can no longer live without. I’m a huge display nerd and the difference has always been significant and obvious to me. When combined with shortened animations, the feeling of smoothness and snappiness is very satisfying.
